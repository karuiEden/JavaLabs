**Скомпилируется ли код в Java 8 без изменений?**

Да.

**Будет ли работать старый Java 7 .class / JAR в Java 8 без перекомпиляции?**

Да.

**Что будет при вызове forEachRemaining?**

Сработает default-метод интерфейса, используя hasNext() и next().

**Что будет при вызове remove()?**

Будет вызван метод твоего класса, который ничего не делает.
UnsupportedOperationException не возникнет.
